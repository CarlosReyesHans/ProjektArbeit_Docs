
A brief of Binary Files :
--------------------------

DIGIO:
------

Dig_8IN_8OUT					- 	Configures LAN9252 in DIG-IO with 8 pins as input and 8 pins as output. 
									
Dig_BIDIR						-	Configures LAN9252 in DIG-IO bidirectional mode.
										

HBI:
-----
									
Indexed_16						-	Configures LAN9252 in HBI - Indexed 16 bit mode.

Indexed_8						-	Configures LAN9252 in HBI - Indexed 8 bit mode.	

Multiplexed_Dual_16				-	Configures LAN9252 in HBI - Multiplexed dual phase 16 bit mode.																		

Multiplexed_Dual_8				-	Configures LAN9252 in HBI - Multiplexed dual phase 8 bit mode.		

Multiplexed_Single_16			-	Configures LAN9252 in HBI - Multiplexed single phase 16 bit mode.			

SPI:
-----

SPI-3PortMode-ExpansionMode		- 	Configures LAN9252 in SPI - 3 port mode /SPI - Expansion mode.

SPI-withGPIO-2PortMode			-	Configures LAN9252 in SPI with GPIO - 2 port mode.
