
A brief of Binary Files :
--------------------------
These files are intended to program PIC32MX available at LAN9252 HBI or SPI EVBs
HBI:
-----
									
HBI_INDEXED_16BIT_XC32_PIC32MX795F512L.hex	- Configures LAN9252 in HBI - Indexed 16 bit mode.

HBI_INDEXED_8BIT_XC32_PIC32MX795F512L.hex	- Configures LAN9252 in HBI - Indexed 8 bit mode.	

HBI_MDP_16BIT_XC32_PIC32MX795F512L.hex - Configures LAN9252 in HBI - Multiplexed dual phase 16 bit mode.																		

HBI_MDP_8BIT_XC32_PIC32MX795F512L.hex -	Configures LAN9252 in HBI - Multiplexed dual phase 8 bit mode.		

HBI_MSP_16BIT_XC32_PIC32MX795F512L.hex - Configures LAN9252 in HBI - Multiplexed single phase 16 bit mode.			

SPI:
-----

SPI_XC32_PIC32MX795F512L.hex - 	Configures LAN9252 in SPI - 3 port mode /SPI - Expansion mode or SPI with GPIO - 2 port mode.
